# car-quality-prediction-random-forest
This project uses the UCI Car Evaluation dataset to predict car quality levels (unacc, acc, good, vgood) using the Random Forest Classifier. All features are categorical and encoded before training. The model is evaluated using accuracy and classification metrics to measure performance..
